# Files Service

Сервис передачи файлов.

## Функциональность

- **Загрузка файла** — сохранение файла с получением URI
- **Получение файла** — загрузка по ранее полученному URI

## API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | /v1/file/new | Загрузить новый файл |
| POST | /v1/file/by-uri | Получить файл по URI |

## Схема данных

- **V1File** — login, filename, content (base64?)
- **V1FileNewResponse** — current_user, uri

## Потенциальные проблемы

- content as string в JSON — неэффективно для бинарных файлов
- Нет размера файла, MIME type
- URI не имеет схемы (неясно s3://, file://, http://)
- Нет авторизации на скачивание (любой current_user может скачать любой файл)
